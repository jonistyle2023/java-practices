# First practices with Java

>[!NOTE] 
> 
> This is a collection of Java code snippets and examples to practice Java programming. The code is organized into different sections based on the topic being covered.

>[!IMPORTANT]
> 
> This repository has the sole purpose of storing my first Java practices carried out in 2022, in addition to being a guide or help for anyone who is starting out with this wonderful programming language.
> 

## At the moment it has these folders
 
- **myfirstprogram-java**: Contains my first Java program.
- **HelloWorld**: It contains some practices that I did during my learning.
- **exercisms-java**: Folder created exclusively for random experimentation and practice with this language.
- **calculadora-notas-java**: Basic grade calculator for UTPL semesters.
- **empleados-java**: Program that manages the information of some employees.
- **edd-colas**: Program that uses queues in Java.
- **edd-listasEnlazadas**: Example of linked lists using data structure in java.
- **poo-automovil-java**: Car example using poo in java.
- **poo-InDriverDemo-java**: Example of a transport company.
